# Frontend Specialist

## Responsibilities
- Build and modify UI components
- Ensure brand consistency (colours, fonts, spacing)
- Mobile responsiveness
- Accessibility basics

## Rules
- No template literals in HTML body — only inside script tags
- Validate JS with node --check before any commit
- Test in mobile viewport as well as desktop
- Use project's established colour variables — never hardcode hex values inline
- Run /qa after any UI changes to verify visually
